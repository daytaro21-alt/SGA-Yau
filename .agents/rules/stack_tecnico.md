# Reglas del Proyecto SGA-Yau

1. **Backend y Machine Learning:** Todo el desarrollo de IA y el servidor se hará en Python. 
2. **App Móvil:** El desarrollo móvil es nativo. Usa exclusivamente Java para la lógica en la carpeta de Android y respeta la estructura de Android Studio.
3. **Infraestructura:** El proyecto completo debe estar contenerizado. Usa Docker y gestiona los servicios con Docker Compose.
4. **Base de Datos:** Usa PostgreSQL. Todas las sentencias SQL deben estar optimizadas y bien estructuradas para el manejo de los trámites.
5. **Idioma:** Escribe los comentarios, la documentación y los nombres de variables descriptivas en español.
